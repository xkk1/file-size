# 文件大小

查看文件大小，转换文件大小工具

## Web

<https://xkk1.github.io/file-size/>

`git clone https://github.com/xkk1/file-size.git`  
打开 `file-size/web/index.html`

## Python

Python 3.8+

```shell
git clone https://github.com/xkk1/file-size.git
cd file-size/python
# 命令行
python file_size.py
# GUI tkinter
pythonw file_size_gui.pyw
```

```
$ python file_size.py
用法：file_size.py [文件]...
显示文件大小信息
```
